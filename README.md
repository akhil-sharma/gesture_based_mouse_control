# Gesture Based Mouse Control
A Python3 application to control the movement of mouse pointer using gestures.

## Dependencies

1. OpenCV3
2. pyautogui
3. numpy
4. Python 3.x

## Executing

- Start the application by running main.py from your terminal using the following command: 
> `python main.py`
